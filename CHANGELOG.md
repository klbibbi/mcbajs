## [1.0.0] - 2025-11-23

### Added
- Initial release
- Tamed dogs poop cocoa beans near players
- Custom poop sound effect
- Configurable scheduler interval (default: 1 minute)
- Configurable poop probability (default: 1/30 chance)
- Configurable cocoa beans amount (default: 2)
- Configurable player distance (default: 50 blocks)
- Only tamed dogs within range of players are affected
- Wild wolves do not poop
- Resource pack with custom sound
- Server-side resource pack configuration support
- GitHub Actions for automated releases

### Technical
- Minecraft 1.21.10 support
- Paper 1.21.10
- Java 21
- GPLv3 License
